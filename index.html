Maze Creator

It is a website, which purpose is drawing and generating mazes and visualization of these steps.

 Jest to stona internetowa, której celem jest rysowanie i generowanie labiryntów oraz wizualizacja tych procesów.

 User is able to draw any shape of board and then pick one of algorithms to generate maze. Afterwards website is displaying process of generation.
Written entirely in pure HTML, CSS and JS. Użytkownik może stworzyć własną planszę poprzez narysowanie komórek - sześciokątów. Następnie można obejrzeć proces, podczas którego
generowany jest labirynt za pomocą wybranego algorytmu.

Chinese Checkers

It is a university assignment, which is game application to play chinese checkers online with friends.

 Jest to aplikacja, przeznaczona do gry w chińskie warcaby (Trylmę) ze znajomymi przez internet.

 User is able to connect to server via IP and play a game with others.
Server is also connected to database to store all games.
It is also possible to see a game from this database. Użytkownik może połączyć się do serwera poprzez IP i zagrać w warcaby z innymi.
Serwer ponadto jest połączony do bazy danych, w której przechowuje informacje o rozgrywkach.
Możliwe jest również obejrzenie gry z bazy.
